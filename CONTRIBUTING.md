# Contributing

Thank you for contributing to HttplibApp.

## Coding Standards

- Write source code comments in **English**.
- Keep comments concise and focused on intent.
- Prefer self-explanatory code and use comments only when they add context that is not obvious from implementation.

## Workflow

1. Fork the repository.
2. Create a branch for your change.
3. Add or update tests for the change.
4. Open a pull request with a clear description.
